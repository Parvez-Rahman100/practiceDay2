Semantic Tags

Semantic tag means it contain meaningful content.A semantic element clearly describes its meaning to both the browser and the developer.In HTML there are some semantic elements that can be used to define different parts of a web page:

Nav Elements :

Nav elements means navigation links. In this element navigation will be putted.

Footer Elements

The footer element defines a footer for a document or section.

Header Elements

The header element represents a container for introductory content or a set of navigational links.

Article Elements

The article element specifies independent, self-contained content. Article elements can be used in Forum Post ,Blog Post,User Comment ,Newspaper Article

Aside Elements

The aside element defines some content aside from the content it is placed in (like a sidebar).
